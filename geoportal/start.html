Geoportal: A Map Displays Geotagged Twitter Data for Indicators of Happiness, Diet, and Physical Activity

About Geoportal

This online mapping application may be used to look up geotagged Twitter data as indicators of Happiness, Diet, and Physical Activity at the state, county, zip code or census tract levels. Data dictionary includes food, excercise, happiness and alcohol variables. *Twitter data collection period: April 2015â€“ March 2016. Data summaries were derived from 80 million tweets from the contiguous United States. Please refer to the following manuscript for more details on the data collection. Map was built with Carto and Google Maps

The specific functions of the Geoportal map are as follows:
Function 1. View data at the clicked location.
Function 2. View data at the searched location.
Function 3. View aggregated data at a drawn region by choosing "Aggregate" button after drawing the region with the tools on the top of the map.
Function 4. Download raw data of state, county, zip code, or census tract in csv/txt format with links in the window.
